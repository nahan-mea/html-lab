Types of Lists

1. Unordered Lists

An unordered list is a collection of items that do not have a specific order. The items are typically marked with bullet points.

Example:

  • numeric
  • capital
  • small

2. Ordered Lists

An ordered list is a collection of items that are presented in a specific sequence, typically numbered.

Example:

  1. circle
  2. disc
  3. square

3. Definition Lists

A definition list is used for a list of terms and their descriptions. It is made up of pairs of terms and their definitions.

Example:

HTML
A markup language for creating web pages.
CSS
A style sheet language used for describing the presentation of a document written in HTML.
